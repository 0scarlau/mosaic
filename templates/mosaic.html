Liquid Studio Mosaic Challenge!

How to use:

Target path - Name of the file you want to be processed to create Mosaic on (Placed in /images/target/ example: a.jpg)

Tile Folder - Name of the folder where the tile images are placed to create the Mosaic image (Placed in /images/tile/ example: flower)

Save Mosaic Image - Name of the mosaic imaged to be saved in /output example: output.jpg

Grid Size - Mosaic image pixel sizes example: 100 100

Resize - Resizing the tile images before processing and building mosaic example: 20 20
